# 🏥 Medical Document Q&A — RAG System

A Retrieval Augmented Generation (RAG) pipeline that answers medical 
queries based on a local knowledge base. No cloud API. Fully local.

## ✨ Features
- Semantic search over medical knowledge using **ChromaDB**
- Local LLM inference via **Ollama** (qwen3.5:4b)
- Structured medical assessment — condition, severity, recommended action
- Fully offline — no OpenAI, no cloud API

## 🛠️ Tech Stack
Python · Ollama · ChromaDB · RAG Pipeline · Local LLM
